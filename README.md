# ממלכת המילים הקסומה

אב־טיפוס משחק דפדפן לילדים בגיל 6-7 לתרגול קריאה והכתבה בעברית ובאנגלית.

המשחק משתמש בעולם קוסמים מקורי, לא במותג קיים, כדי להימנע מסיכוני זכויות יוצרים וסימני מסחר. הילד/ה זז/ה במפה עם מקשי המקלדת ונכנס/ת למשימות קצרות של גרירת אותיות והקלדת מילים.

## הרצה

אפשר לפתוח ישירות את `index.html` בדפדפן.

אפשר גם להריץ שרת מקומי:

```bash
python3 -m http.server 8000
```

ואז לפתוח:

```text
http://localhost:8000
```

## מה יש באב־טיפוס

1. מפה קטנה בסגנון RPG דו־ממדי.
2. תנועה עם מקשי חיצים או W A S D.
3. כניסה למשימת למידה עם רווח.
4. שלושה אזורי למידה: עברית, ניקוד, אנגלית.
5. גרירת אותיות או לחיצה עליהן לבניית מילה.
6. הקלדת מילה כחלופה לגרירה.
7. משוב מיידי ורמז לימודי.
8. מערכת כוכבים ושערים בסיסית.

## אוצר מילים ראשוני

עברית:

- דָּג
- בַּיִת
- שָׁלוֹם
- סֵפֶר

אנגלית:

- cat
- sun

## עקרונות למידה

- משימה קצרה אחת בכל פעם.
- פירוק מילה לחלקים קטנים.
- עברית: אות + ניקוד + הברה + מילה.
- אנגלית: phonics בסיסי, במיוחד CVC words.
- טעות מובילה לרמז ולא לעונש.

## מגבלות ידועות

- אין עדיין שמירת התקדמות.
- אין עדיין הקראה קולית.
- אין עדיין התאמה אישית לפי תחומי עניין.
- אין עדיין בדיקות אוטומטיות.
- תמיכת ניקוד תלויה במקלדת ובדפדפן; לכן גרירה חשובה במיוחד.
