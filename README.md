# ממלכת המילים הקסומה

אב־טיפוס משחק דפדפן בתלת־ממד לילדים בגיל 6-7 לתרגול קריאה והכתבה בעברית ובאנגלית.

המשחק משתמש בעולם קוסמים מקורי, לא במותג קיים, כדי להימנע מסיכוני זכויות יוצרים וסימני מסחר. הילד/ה זז/ה במפה עם מקשי המקלדת ונכנס/ת למשימות קצרות של גרירת אותיות והקלדת מילים.

העיצוב הנוכחי שואב השראה כללית ממשחקי למידה צבעוניים לילדים, עם מפה בהירה, תחנות, שביל התקדמות ומשוב ברור. אין שימוש בנכסים, דמויות, לוגו או מסכים ממותגים של מוצר אחר.

## הרצה

אפשר לפתוח ישירות את `index.html` בדפדפן.

אפשר גם להריץ שרת מקומי:

```bash
python3 -m http.server 8000
```

ואז לפתוח:

```text
http://localhost:8000
```

## בדיקות

בדיקת התוכן מוודאת שאין מזהי משימות חסרים, שדות חסרים או כפילויות בסיסיות:

```bash
node tests/validate-content.js
```

## מה יש באב־טיפוס

1. חצר משחק קטנה בסגנון RPG תלת־ממדי.
2. חצר תלת־ממדית בסיסית עם טירה, תחנות למידה, אותיות במרחב ודמות מקורית.
3. תנועה עם מקשי חיצים או W A S D.
4. כניסה למשימת למידה עם רווח.
5. אזורי למידה: עברית, ניקוד, אנגלית, סיפור והכתבה.
6. אזור סיפור והכתבה עם הקראה קולית דרך הדפדפן.
7. גרירת אותיות או לחיצה עליהן לבניית מילה.
8. הקלדת מילה כחלופה לגרירה.
9. משוב מיידי ורמז לימודי.
10. מערכת כוכבים ושערים בסיסית.

## אוצר מילים ראשוני

עברית:

- דָּג
- בַּיִת
- שָׁלוֹם
- סֵפֶר
- סיפור קצר: דָּן רָאָה דָּג. הַדָּג שָׂחָה בַּיָּם.

אנגלית:

- cat
- sun
- Short story: A cat sat on a mat. The cat is happy.

## עקרונות למידה

- משימה קצרה אחת בכל פעם.
- פירוק מילה לחלקים קטנים.
- עברית: אות + ניקוד + הברה + מילה.
- אנגלית: phonics בסיסי, במיוחד CVC words.
- טעות מובילה לרמז ולא לעונש.

## מגבלות ידועות

- אין עדיין שמירת התקדמות.
- יש הקראה קולית בסיסית דרך הדפדפן, אבל איכות הקול תלויה במערכת ההפעלה ובקולות המותקנים.
- יש אב־טיפוס תלת־ממדי ראשוני, אבל הדמות עדיין placeholder ולא עיצוב סופי.
- אין עדיין התאמה אישית לפי תחומי עניין.
- יש בדיקת תוכן בסיסית, אבל אין עדיין בדיקות מלאות ללוגיקת המשחק.
- תמיכת ניקוד תלויה במקלדת ובדפדפן; לכן גרירה חשובה במיוחד.
